Its velocity $v$ as a function of position $x$ is continuously given by the equation $v(x) = \frac{\alpha}{x}$, where $\alpha$ is a positive constant. Assuming the particle is located at $x = x_0$ at time $t = 0$, which of the following expressions correctly represents the **acceleration** of the particle as a function of **time** $t$? (A) $a(t) = -\frac{\alpha}{t^2}$ (B) $a(t) = -\frac{\alpha^2}{x^3}$ (C) $a(t) = -\frac{\alpha^2}{(2\alpha t + x_0^2)^{3/2}}$ (D) $a(t) = \frac{\alpha}{(2\alpha t + x_0^2)^{1/2}}$ (E) $a(t) = -\frac{\alpha^2}{(\alpha t + x_0^2)^3}$ ##### **Expert Calculus Explanation & Distractor Analysis** **Correct Answer: (C)** This problem demands a multi-step calculus synthesis. You cannot simply take the derivative of $v$ with respect to $t$ directly because $v$ is given in terms of $x$. You must first find the acceleration as a function of position using the chain rule, and then solve the separable differential equation to map position $x$ to time $t$. **Step 1: Find acceleration as a function of position, $a(x)$.** Using the chain rule for velocity dependent on position: $$a = \frac{dv}{dt} = \frac{dv}{dx} \frac{dx}{dt} = v \frac{dv}{dx}$$ Substitute $v(x) = \alpha x^{-1}$: $$a(x) = \left(\frac{\alpha}{x}\right) \left(-\alpha x^{-2}\right) = -\frac{\alpha^2}{x^3}$$ *(Note: If the question asked for $a(x)$, (B) would be correct. But we need $a(t)$).* **Step 2: Solve the differential equation to find $x(t)$.** Start with the definition of velocity: $$v = \frac{dx}{dt} = \frac{\alpha}{x}$$ Separate the variables and integrate from $t=0$ to $t$: $$\int_{x_0}^{x} x \, dx = \int_{0}^{t} \alpha \, dt$$ $$\frac{1}{2}x^2 - \frac{1}{2}x_0^2 = \alpha t$$ $$x^2 = 2\alpha t + x_0^2 \implies x(t) = \sqrt{2\alpha t + x_0^2}$$ **Step 3: Substitute $x(t)$ into $a(x)$ to find $a(t)$.** $$a(t) = -\frac{\alpha^2}{(x(t))^3} = -\frac{\alpha^2}{\left(\sqrt{2\alpha t + x_0^2}\right)^3} = -\frac{\alpha^2}{(2\alpha t + x_0^2)^{3/2}}$$ --- ### Part 2: Free Response Question (FRQ) **Prompt:** An experimental maglev cart moves along a straight horizontal track.
